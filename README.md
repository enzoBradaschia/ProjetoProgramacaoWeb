🍔🍕 O Estradeiro - Pizzaria & Hamburgueria

Bem-vindo ao repositório do site O Estradeiro! Esta é uma Landing Page desenvolvida para apresentar o cardápio, a história e as informações de contato de um restaurante focado em refeições caseiras, pizzas artesanais e hambúrgueres.

🔗 Acesso ao Projeto

O site está hospedado e disponível online através do GitHub Pages. Você pode acessá-lo clicando no link abaixo:

👉 https://enzobradaschia.github.io/ProjetoProgramacaoWeb/

🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias:

HTML5: Estruturação semântica do conteúdo.

CSS3: Estilização da página utilizando Flexbox para garantir um layout responsivo e moderno sem a necessidade de frameworks externos.

Google Fonts: Utilização da fonte Roboto para uma tipografia limpa e legível.

Google Maps Embed API: Integração de mapa interativo no rodapé para mostrar a localização exata do restaurante.

✨ Funcionalidades

Design Responsivo: O site se adapta perfeitamente a diferentes tamanhos de tela (celulares, tablets e desktops).

Navegação Suave (Smooth Scroll): Ao clicar nos links do menu superior (Início, Produtos, Sobre Nós, Contato), a tela desliza suavemente até a seção desejada.

Seções Integradas:

Sobre Nós: Apresentação da identidade visual e missão do restaurante.

Produtos: Destaque para os principais pratos (pizzas e hambúrgueres).

Rodapé Completo: Informações de contato (endereço e telefone), horário de funcionamento, links para redes sociais (Instagram e Facebook), link direto para pedidos no iFood e um mapa interativo.

🚀 Como rodar o projeto localmente

Se você deseja baixar e rodar este projeto no seu próprio computador, siga os passos abaixo:

Clone este repositório:

git clone [https://github.com/enzoBradaschia/ProjetoProgramacaoWeb.git](https://github.com/enzoBradaschia/ProjetoProgramacaoWeb.git)


Entre na pasta do projeto:

cd ProjetoProgramacaoWeb


Certifique-se de que as imagens do projeto (logo.jpg, pizza.jpg e hamburguer.jpg) estão dentro de uma pasta chamada img.

Abra o arquivo index.html em qualquer navegador web de sua preferência.

📁 Estrutura de Pastas Esperada

/
├── index.html        # Arquivo principal da página
└── img/              # Pasta contendo os recursos visuais
    ├── logo.jpg
    ├── pizza.jpg
    └── hamburguer.jpg
